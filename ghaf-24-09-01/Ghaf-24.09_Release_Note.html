Introduction

Ghaf Framework is an open-source project for enhancing security through compartmentalization on edge devices.

Release Tag

https://github.com/tiiuae/ghaf/releases/tag/ghaf-24.09

Supported Hardware

  • NVIDIA Jetson AGX Orin
  • NVIDIA Jetson Orin NX
  • Generic x86 (PC)
  • Polarfire Icicle Kit
  • Lenovo ThinkPad X1 Carbon Gen 11
  • Lenovo ThinkPad X1 Carbon Gen 10
  • NXP i.MX 8M Plus

What is new in Ghaf-24.09

Lenovo X1 Carbon Gen 10/11:

  • Trusted Business VM with firewall protection containing the following apps: Microsoft365, Outlook, Teams, Trusted Browser, Text Editor, Video Editor
  • Integrated camera supported with Business VM applications
  • Previous Element-VM modified to more generic Comms-VM adding Slack
  • GlobalProtect VPN client
  • Centralized logging solution using Grafana
  • ZFS file system and logical volume manager
  • Storage VM using impermanence framework
  • USB hot plug supporting input, audio and media devices
  • USB camera support on Chromium VM
  • Initial version of file manager
  • Hardware detection scanner to generate hardware definition files for different laptops
  • GPU acceleration enabled
  • Yubikey authentication
  • Falcon LLM
  • Greetd login manager with system autolock enabled, locks screen after 5 minutes of inactivity
  • UI waybar replaced by EWW taskbar
  • Magnification app
  • Calculator
  • Sticky notes app
  • Screenshot app
  • AppFlowy has been disabled

NVIDIA Jetson Orin NX:

  • Jetpack baseline software updates and fixes.
  • Further refactoring and modularization of the Ghaf framework.
  • Development, testing, and performance tooling improvements.
